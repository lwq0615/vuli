# Vui组件库
 
### 快速开始

### 1.安装组件库
```bash
npm install vui
```

### 2.引用组件库

#### 全部引入
```js
//main.js
import 'vui/dist/css/index.css'
import LuckUI from 'vui'
Vue.use(LuckUI)
```

#### 按需引入
```js
//main.js
import 'vui/dist/css/index.css'
import { button } from 'vui'
Vue.use(button)
```
